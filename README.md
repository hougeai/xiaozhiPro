三步走：

1. 下载自己开发板对应的固件包，并通过烧录工具烧录进设备（后缀为.bin的文件）
   - 参考：[Flash工具/Web端烧录固件（无IDF开发环境）](https://ccnphfhqs21z.feishu.cn/wiki/Zpz4wXBtdimBrLk25WdcXzxcnNS)
